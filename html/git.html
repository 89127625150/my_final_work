Git

  • git status -"Какие файлы находятся в индексе и готовы к коммиту" и "новые файлы которые появились в проекте и пока не отслеживаются Git".
  • git checkout - "используется для переключения между ветками" или "восстановленя файлов в рабочем каталоге".
  • git rev-parse --show-toplevel -команда в системе контроля версий Git выводит абсолютный путь к корневому каталогу репозитория.
  • git init - создание нового репозитория или получение его по существующему URL-адресу.
  • git mv -одновременно переименовать или переместить файл и автоматически обновить индекс.
